# Bloque 01 — 01_liveGallery

> **Fecha:** 14/06/2026  
> **Objetivo:** Galería de 6 productos con zoom en imagen

Galería que muestra productos en cards. Cada card es un **Twig Component** (`ImageCard`). Al hacer click en la imagen, se aplica un **zoom** vía **Stimulus**. Todo renderizado desde PHP, sin React/Vue.

**Stack:** Symfony 7 + Twig Components + Stimulus + AssetMapper + SQLite
