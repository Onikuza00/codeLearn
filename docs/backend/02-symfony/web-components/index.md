# Web Components de Symfony

> **Fecha inicio:** 13/06/2026
> **Estado:** En curso

Los Web Components de Symfony son **bloques de UI reutilizables** que se renderizan desde PHP. No confundir con los Web Components del navegador (Custom Elements, Shadow DOM).

| Concepto | Traducción |
|----------|-----------|
| `<twig:ProductCard>` | "Symfony, renderizá el componente ProductCard acá" |
| La clase PHP | "Symfony, esta es la lógica del componente" |
| El template Twig | "Symfony, este es el HTML del componente" |
