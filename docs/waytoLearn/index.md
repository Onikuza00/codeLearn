# 🧪 Way to Learn

> Ejercicios prácticos de maquetación. Cada ejercicio tiene un recordatorio rápido, un enunciado claro y criterios de evaluación. Sin ayuda, sin teoría nueva — solo práctica.

---

## Cómo funciona

1. Elegí un ejercicio
2. Leé el **recordatorio** (🧠 5 min)
3. Hacé el ejercicio sin ayuda (⏱ cronometrado)
4. Revisamos juntos los errores
